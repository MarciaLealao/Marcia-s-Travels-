# Marcia's Travels

Welcome to the travel photo collection repository! This is where I share all my travel memories captured through images.

## Instructions for Adding Images

1. **Organize your images**: Before uploading, please organize your images into folders named after the location or event (e.g., `Paris_2026`, `Beach_Vacation`).
2. **File Names**: Use clear and descriptive file names for your images (e.g., `Eiffel_Tower_Sunset.jpg`). 
3. **Upload**: To add your images, you can drag and drop them into the appropriate folder in this repository or use the Git command line to push your folders and files directly.
4. **PRs**: If you're collaborating, please submit a pull request with your additions, and I'll review your changes!

Enjoy exploring travel through photos!
